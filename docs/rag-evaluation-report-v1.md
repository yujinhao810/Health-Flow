# HealthFlow RAG 标准评测报告 v1

评测日期：2026-07-20

## 结论

内置公共知识库从 7 篇单 Chunk 文档升级为 7 篇、21 个语义 Chunk，并补充来源 URL、版本、适用范围、编审状态和复核周期。配合 Embedding 每批 10 条、RRF、`gte-rerank-v2` 和 0.165 公共无答案门控，标准集上的 Recall@3、Hit@1、无答案准确率和 Top-1/拒答综合准确率均达到 **100.0%**。

本结果只对当前锁定的 7 篇公共知识文档和 72 道工程评测题有效。正例最低分 0.1703、无答案最高分 0.1649，虽然当前可分，但间隔只有 0.0054；不能把 100% 外推为生产环境中的绝对准确率。

## 语料升级

- 文档数保持 7 篇，沿用原有标题，维持完整文档级相关性标注。
- Chunk 从 7 个增加到 21 个，每篇拆成 3 个职责单一的语义块。
- 为口语表达和第三方危机场景补充关键词，例如“朋友说今晚不想活”“已经准备好了东西”“不要独处”和“代为求助”。
- 每篇文档增加 `sourceUrl`、`version`、`audience`、`evidenceScope`、`editorialStatus`、`lastReviewedAt` 和 `nextReviewDueAt`。
- 明确标记为一般安全教育和就医分流资料，不用于诊断、处方或个体化治疗；正式医疗用途前仍需临床专业人员复核。
- Embedding 改为每批最多 10 条，避免 DashScope 批量上限导致文档向量回退后与查询向量模型不一致。

## 评测范围

- 固定语料：7 篇内置公共知识文档，共 21 个 Chunk。
- 标准问题：72 道，其中 58 道有答案、14 道无答案。
- 问题类型：22 道语义改写、14 道口语表达、13 道直接问法、9 道多文档问题、14 道无答案问题。
- 标注方式：对全部 7 篇文档进行完整 0-3 级相关性判断；同文档多个 Chunk 在评测时按文档去重。
- 检索流程：词法召回 + Embedding 语义召回 → RRF → `gte-rerank-v2` → 公共证据门控。
- 数据边界：真实模型评测只发送公开问题和公共知识片段，没有发送用户上传文件或健康记录。

| 指标 | 上一版知识库 | 新版知识库 | 变化 |
| --- | ---: | ---: | ---: |

| Hit@3 | 98.3% | 100.0% | +1.7 pp |
| Recall@3 | 98.3% | 100.0% | +1.7 pp |
| Precision@3 | 42.5% | 43.1% | +0.6 pp |
| MRR | 0.974 | 1.000 | +0.026 |
| MAP@5 | 0.968 | 0.997 | +0.029 |
| nDCG@5 | 0.972 | 0.995 | +0.023 |
| 无答案准确率 | 92.9% | 100.0% | +7.1 pp |
| Top-1/拒答综合准确率 | 95.8% | 100.0% | +4.2 pp |
| 端到端 P95 | 572 ms | 473 ms | -99 ms |

当前评测集大多数问题只有 1-2 篇相关文档，因此 Precision@3 的理论上限约为 43.1%；新版达到该上限。返回结果 Precision@5 为 39.9%，因为系统会保留同一相关文档中的互补 Chunk，而评测标注和该指标按文档去重计算。

## 门控校准

- 正例 Rerank TopScore：最小值 0.1703，P10 0.1850，P50 0.3776。
- 无答案 TopScore：P50 0.0426，P90 0.1552，最大值 0.1649。
- 阈值 0.15：正例通过率 100.0%，无答案拒绝率 85.7%，综合准确率 97.2%。
- 阈值 0.165：正例通过率 100.0%，无答案拒绝率 100.0%，综合准确率 100.0%。
- 阈值 0.20：正例通过率下降到 77.6%，不采用。

默认 `RAG_PUBLIC_RERANK_MIN_SCORE` 已调整为 0.165。门控只应用于公共知识片段，不提高用户个人文档的最低分数。

## 性能

- 真实 Rerank：72/72 次成功应用。
- Rerank 延迟：P50 154 ms，P95 198 ms。
- 完整检索延迟：P50 346 ms，P95 473 ms。
- 公共文档 Embedding 按模型和语料更新时间缓存在 API 进程内；查询向量每轮计算。
- 远程 Embedding 每批最多 10 条，批次模型或向量格式不一致时整批安全回退。
- Embedding 或 Rerank 失败时，自动回退到词法 RRF，不使用不兼容的语义向量。

## 剩余风险

- 当前没有漏召回正例或无答案误召回，但无答案样本只有 14 道，95% Wilson 置信区间仍为 78.5%-100.0%。
- 门控分数间隔较窄，模型版本、语料微调或新问题分布都可能改变最佳阈值；语料更新后必须重新评测。
- 资料由工程侧编审并链接外部参考来源，尚未经过独立临床专业人员逐条审核。
- 当前语料聚焦危机、心理边界、焦虑稳定、睡眠、运动安全、用药边界和专业求助，不覆盖具体疾病诊疗、儿童剂量、孕产、疫苗、检验值解读或个体化营养方案。
