# HealthFlow RAG 标准评测报告 v1

评测日期：2026-07-17

## 结论

加入公共知识库 Embedding 语义召回、RRF 融合、`gte-rerank-v2` 和 0.15 无答案门控后，宏平均 Recall@3 从 **57.8%** 提升到 **98.3%**，Top-1/拒答综合准确率从 **66.7%** 提升到 **95.8%**。

无答案准确率为 **92.9%**。72 道题中剩余 1 道正例漏召回和 1 道无答案误召回，二者位于 Rerank 分数重叠区。当前结果对锁定的 7 篇公共知识文档有效，不能直接外推到任意用户上传文档。

## 评测范围

- 固定语料：7 篇内置公共知识文档，每篇 1 个 chunk。
- 标准问题：72 道，其中 58 道有答案、14 道无答案。
- 问题类型：22 道语义改写、14 道口语表达、13 道直接问法、9 道多文档问题、14 道无答案问题。
- 标注方式：对全部 7 篇语料进行完整 0-3 级相关性判断；未列出的文档均为 0 级不相关。
- 优化流程：词法召回 + Embedding 语义召回 → RRF → `gte-rerank-v2` → 公共证据门控。
- 数据边界：仅测试内置公共知识库，没有向外部服务发送用户上传文件。

## 前后对比

| 指标 | 原词法 RRF | 优化后 | 变化 |
| --- | ---: | ---: | ---: |
| Hit@1 | 62.1% | 96.6% | +34.5 pp |
| Hit@3 | 63.8% | 98.3% | +34.5 pp |
| Recall@3 | 57.8% | 98.3% | +40.5 pp |
| Precision@3 | 24.1% | 42.5% | +18.4 pp |
| 返回结果 Precision@5 | 47.8% | 25.5% | -22.3 pp |
| MRR | 0.626 | 0.974 | +0.348 |
| MAP@5 | 0.570 | 0.968 | +0.398 |
| nDCG@5 | 0.582 | 0.972 | +0.390 |
| 无答案准确率 | 85.7% | 92.9% | +7.2 pp |
| Top-1/拒答综合准确率 | 66.7% | 95.8% | +29.1 pp |
| 端到端 P95 | 3 ms | 572 ms | +569 ms |

优化后 Hit@3 的 95% Wilson 置信区间为 90.9%-99.7%。无答案准确率的 95% 置信区间为 68.5%-98.7%；无答案样本仍需扩充。

`返回结果 Precision@5` 下降是因为广召回会固定提供更多候选，而当前每题只标注 1-2 篇相关文档。最终回答使用的 TopK 通常小于 5，且排序指标 MRR、MAP 和 nDCG 均显著提升。

## 门控校准

正例 Rerank TopScore：最小值 0.1400，P10 0.1672，P50 0.1966。

无答案 TopScore：P50 0.0132，P90 0.1488，最大值 0.1614。

阈值 0.15 时，正例通过率 98.3%，无答案拒绝率 92.9%，综合准确率 95.8%，是当前标准集上的最佳平衡点。门控只应用于内置公共知识文档，不提高用户个人文档的最低分数。

## 性能

- 真实 Rerank：72/72 次成功应用。
- Rerank 延迟：P50 136 ms，P95 171 ms。
- 完整检索延迟：P50 331 ms，P95 572 ms。
- 公共文档 Embedding 按模型和语料更新时间缓存在 API 进程内；查询向量每轮计算。
- Embedding 或 Rerank 失败时，自动回退到原词法 RRF，不使用广召回结果。

## 剩余失败

- 正例漏召回：`crisis-02`，相关文档最高分略低于 0.15。
- 无答案误召回：`negative-13`，儿童退烧药剂量问题最高分高于 0.15。

这两个样本的分数区间重叠，继续只调整单一阈值会在召回率和无答案准确率之间互相损失。下一步应增加独立无答案分类器或扩大人工标注集后再做分类型阈值。

## 适用边界

本报告的指标对当前锁定的 7 篇公共语料有效。由于语料规模较小、问题由工程侧标注，结果不能替代独立测试人员或医疗专业人员的盲审，也不能代表用户 PDF、OCR 文档和大规模知识库的表现。

标准集会校验数据库公共语料标题；语料发生增删时评测器会拒绝出分，要求重新完成相关性标注，避免 Precision 因漏标而失真。
